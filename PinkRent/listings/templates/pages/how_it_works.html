{% trans "How does item rental work?" %}

{% trans "The owner uploads a listing with the item for rent. The Pink Pink team reviews your listing. If it is approved, you will receive an email confirming the approval of the listing to your specified email address." %}

{% trans "How does the submission/receipt of a rental order work?" %}

{% trans "The tenant selects a rental period on the calendar and clicks Reserve The owner can confirm the order upon receiving a request to rent their item. After confirmation, the tenant receives a notification that the order has been accepted." %}

{% trans "How does payment for an order work?" %}

{% trans "The owner pays for the order in the My Profile section, under the Orders tab. They enter their details and the address to which the order will be sent. The owner sends the rental payment to Pink Pink's payment system using the specified payment details." %}

{% trans "What percentage of the owner's earnings does Pink Pink keep?" %}

{% trans "10% of the profit from one rented listing goes to Pink Pink as a service fee for platform usage. This portion is automatically deducted from your earnings when the owner pays the rental fee." %}

{% trans "How does the order get sent to the tenant?" %}

{% trans "After payment for the rental, Pink Pink sends a message to the tenant with the address of the parcel locker. The tenant then sends the order themselves." %}

{% trans "How does order receipt work?" %}

{% trans "Upon receiving the order, the tenant sends a confirmation to the UPUPAM email (lapinkpinkstudio@gmail.com)." %}

{% trans "How does the return of the item to the owner work?" %}

{% trans "After the rental period ends, the tenant receives the parcel locker address and sends the item back independently." %}

{% trans "When does the owner receive payment for the order from the tenant?" %}

{% trans "When the owner confirms receiving the returned item, the money for the rental is sent to the owner's account." %}

{% trans "What items can be rented on the UPUPAM platform?" %}

{% trans "On the UPUPAM platform, you can rent evening dresses and shoes. Counterfeit items are prohibited." %}

{% trans "Who is responsible for cleaning the item?" %}

{% trans "The item owner - the tenant - is responsible for cleaning the item after it is returned by the tenant." %}

{% trans "What should I do if the tenant returns a damaged item?" %}

{% trans "Please take photos of the rented item within 24 hours before sending it back to the tenant." %}

{% trans "What should I do if the tenant claims that I returned a damaged item?" %}

{% trans "Please take photos of the rented item before sending it back to the tenant." %}

{% trans "Who covers the losses for a damaged item?" %}

{% trans "If, upon return from the tenant, the item is damaged in a condition different from the photos taken by the tenant before sending it, and the owner does not have photos of the item taken before returning it, showing otherwise, then the owner is responsible for covering the losses." %}
